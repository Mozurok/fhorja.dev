# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Summary

In short:

- Be respectful and constructive.
- Focus on technical merit, not on the contributor.
- Disagree without being disagreeable.
- Harassment, discrimination, and personal attacks are not tolerated.

## Reporting

If you witness or experience behavior that violates the Contributor Covenant in any space connected to this project (issues, PRs, discussions, social media, etc.), report it to the maintainer via the email address listed on the GitHub profile.

Reports will be reviewed in good faith. Given the project's BDFL solo-maintainer model, enforcement decisions are made by the maintainer; there is no committee or appeals process beyond that.

The maintainer commits to:

- Treating reports confidentially within reason.
- Responding to reports on a best-effort timeline.
- Taking action proportional to the violation, ranging from a warning to permanent ban from project spaces.
